## 开发流程

1. clone ant-design-plus 的代码到本地

```
git clone git@github.com:alitajs/ant-design-plus.git
```

2. 安装项目依赖，推荐使用yarn

```
yarn
```

3. 使用yarn安装每个package下的依赖

```
yarn bootstarp

```

4. 在本地运行 Ant Design Plus 的网站

```
yarn start
```
